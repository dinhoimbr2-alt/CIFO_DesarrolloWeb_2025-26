
# Plantilla Proyecto Web

Estructura mínima para proyectos del curso CIFO.

## Cómo ejecutar
- Abre `index.html` en el navegador **o** usa la extensión *Live Server* de VS Code.

## Estructura
- `index.html` — estructura base
- `style.css` — estilos
- `script.js` — interacción
