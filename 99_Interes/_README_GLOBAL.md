
# CIFO Desarrollo Web 2025 — Estructura Profesional (Windows)

Esta carpeta organiza todo el curso **IFCD0210** por módulos:
- `01_MF0491_Cliente` — HTML, CSS, JS, Accesibilidad, Multimedia.
- `02_MF0492_Servidor` — Lado servidor, BD, SQL, seguridad.
- `03_MF0493_Implantacion` — Despliegue, servidores, documentación.
- `04_Practicas` — Material e informe de prácticas no laborales.
- `05_Portafolio` — Proyecto final y material para entrevistas.
- `06_GitHub_Portfolio` — Copias *limpias* para publicar en GitHub.
- `99_Extras` — Certificados, temarios y plantillas.

## Recomendaciones de uso
- Toma notas en **Markdown (.md)** dentro de `Apuntes`.
- Cada **proyecto** en su propia carpeta con `README.md`.
- Haz **backup** semanal a la nube (Drive/OneDrive) y a GitHub.
- Mantén una convención visual en notas: **Concepto, Ejemplo, Tip, Error, Resumen**.
