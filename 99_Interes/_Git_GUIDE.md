
# Guía rápida Git + GitHub (flujo simple)

```bash
# 1) Inicializa repo en un proyecto
git init
git add .
git commit -m "Primer commit"

# 2) Crea repositorio en GitHub y conecta
git remote add origin https://github.com/USUARIO/NOMBRE_REPO.git
git branch -M main
git push -u origin main
```

## Notas
- Añade un `README.md` claro en cada proyecto.
- No subas `Certificados` ni datos personales.
