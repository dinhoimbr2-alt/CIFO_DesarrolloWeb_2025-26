
# Atajos útiles en Windows + VS Code

- Abrir terminal en carpeta: **Shift + clic derecho** → "Abrir ventana de PowerShell aquí"
- VS Code: `Ctrl+Shift+P` (Comandos), `Ctrl+\`` (Terminal integrada), `Ctrl+P` (Go to file)
- Markdown preview en VS Code: `Ctrl+Shift+V`

## Sugerencia de backup semanal
1. Comprime cada módulo en `Drive\CIFO_Backup\{modulo}.zip`
2. Sube proyectos a GitHub (ver `_Git_GUIDE.md`).
